# runner-vue

A Vue 3 + Vite project with comprehensive testing setup using Jest and Vue Testing Library. This project includes test files generated by various AI models for research purposes.

## Prerequisites

- Node.js ^20.19.0 || >=22.12.0
- npm or yarn

## Project Setup

```sh
npm install
```

## Available Scripts

### Development
```sh
npm run dev          # Start development server with hot-reload
npm run preview      # Preview production build locally
```

### Build
```sh
npm run build        # Build for production
```

### Testing
```sh
npm run test         # Run Jest tests
```

### Code Quality
```sh
npm run lint         # Lint and fix with ESLint
npm run format       # Format code with Prettier
```

## Project Structure

```
src/
├── App.vue              # Main application component
├── 00_component.vue     # Base component for testing
├── main.js              # Application entry point
└── *.test.js            # Test files generated by different AI models:
    ├── claude_run*.test.js
    ├── gemini_run*.test.js
    ├── gpt4o_run*.test.js
    ├── grok_run*.test.js
    └── llama_run*.test.js
```

## Tech Stack

- **Framework**: Vue 3.5.22
- **Build Tool**: Vite 7.1.11
- **Testing**: Jest 29.7.0 + Vue Testing Library 8.1.0
- **Linting**: ESLint 9.37.0
- **Formatting**: Prettier 3.6.2

## Testing Framework

This project uses Jest with Vue Testing Library for component testing. The test environment is configured with:

- **Jest Environment**: jsdom
- **Vue Test Utils**: 2.4.6
- **Vue3 Jest**: 29.2.6
- **Testing Library Jest DOM**: 6.9.1
